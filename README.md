## Ts-React-Demo

### 技术栈：

* TypeScript
* React
* Redux
* webpack

### 项目目录

```
|-- /config/        webpack配置文件
|-- /public/        一些公共资源
|-- /scripts/       create-react-app自动生成的一些脚本
|-- /src/           项目主目录
|-- tsconfig.json   TypeScript配置文件
|-- tslint.json     tslint配置文件，用于代码检查
```

### 使用

克隆该项目到本地：

`$ git clone git@github.com:LucaslEliane/ts-react.git`

执行依赖的安装工作并且运行：

* yarn:

`$ yarn`

`$ yarn start`

* npm

`$ cnpm install`

`$ npm start`

访问`localhost:3000`。

其余脚本见`package.json`
